1. The Integers

DEFINITION 1.1. Let $a,b\in\mathbb{Z}$. We say that "$b$ divides $a$", or "$b$ is a divisor of $a$", or "$a$ is a multiple of $b$", and write $b|a$, if there is an integer $c$ such that $a=bc$.

LEMMA 1.2. If $b|a$ and $a\ne0$, then $|b|\le|a|$.
Proof. Indeed, by definition of divisibility we have $a=bc$ for some integer $c$; in particular, both $b$ and $c$ are nonzero since $a\ne0$. Since $c\in\mathbb{Z}$ and $c\ne0$, we have $|c|\ge{1}$. And then $$|a|=|b|\cdot|c|\ge{|b|}\cdot{1} = |b|$$ as claimed.

"Divisibility" defines an order relation on the set of nonnegative integers. What Lemma 1.2 says is that, to some extent, this "new" order relation is compatible with the ordinary one $\le$.

FACT (Well-ordering principle). Every nonempty set of nonnegative integers contains a least element.

THEOREAM 1.2. (Division with remainder) Let $a,b$ be integers, with $b\ne0$. Then there exist a unique $q\in\mathbb{Z}$ and a unique $r\in\mathbb{Z}$ such that $a=bq+r$ with $0\le{r}\lt|b|$.
Proof. We can assume that $b\gt{0}$. Indeed, if $b\lt{0}$, we can apply the statement to $-b\gt{0}$ and then flip the sign of $q$ after the fact. Switching the sign of $b$ does not change $|b|$, so the condition on $r$ is unchanged.
Assume then that $b\gt{0}$, and consider all integer linear combinations of $a$ and $b$ of the form $a-bx$ with $x\in\mathbb{Z}$. Note that there are nonnegative integers of this type: if $a$ is itself nonnegative, then $a-b\cdot0=a$ is nonnegative; and if $a$ is negative, then $a-ba=a(1-b)$ is nonnegative because $1-b\le0$. Therefore, the set $$S=\{a-bx\ |\ x\in\mathbb{Z}: a-bx\ge0\}$$ is a nonempty set of nonnegative integers.
By the well-ordering principle, it contains a least element $r$: that is, there is some $x=q$ such that $r=a-bq\ge0$ is smaller than any other nonnegative number of the form $a-bx$. And these $q$ and $r$ are the unique numbers whose existence is claimed in the statement.
Indeed: by construction we have that $a=bq+r$ and that $r\ge0$; we have to verify that (1) $r\lt{b}$ (note that $|b|=b$ since we are assuming $b\gt0$) and (2) the numbers $q$ and $r$ are unique with these properties.
For (1), argue by contradiction. If we had $r\ge{b}$, then we would have $r-b\ge0$; and $$0\le{r-b}=(a-bq)-b=a-b(q+1).$$ This would show that $r-b$ is an element of $S$, since it is a nonnegative integer linear combination of $a$ and $b$. But that is a contradiction, since $r-b\lt{r}$ while $r$ was chosen to be the least element of $S$. Since $r\ge{b}$ leads to a contradiction, we can conclude that $r\lt{b}$ as we claimed.
For (2), assume that $q',r'$ are also integers satisfying the requirement spelled out in the theorem: that is, $a=bq'+r'$, and $0\le{r'}\lt{b}$. We will show that necessarily $q'=q$ and $r'=r$ (this is what we mean by stating that $q$ and $r$ are "unique").
Since $a=bq+r=bq'+r'$, we have $$b(q-q')=r'-r.\qquad\qquad\qquad(1.1)$$
Now, since both $r$ and $r'$ are in the interval $[0, b-1]$, their difference cannot exceed $b-1$. In other words, $|r'-r|\le{b-1}$, and in particular $|r'-r|\lt|b|$. But (1.1) shows that $b$ is a divisor of $r'-r$; if $r'-r\ne0$, then by Lemma 1.2 we would necessarily have $|b|\le|r'-r|$, contradicting the fact that $|b|>|r'-r|$ we observed a moment ago. The only possibility then is that $r'-r=0$. This shows that $r'=r$, and further $b(q-q')=0$. Since $b\ne0$, this implies that $q-q'=0$, and it follows that $q'=q$.
