1. The Integers

DEFINITION 1.1. Divisor.
Let $a,b\in\mathbb{Z}$. We say that "$b$ divides $a$", or "$b$ is a divisor of $a$", or "$a$ is a multiple of $b$", and write $b|a$, if there is an integer $c$ such that $a=bc$.

Keep in mind that $0$ only divides $0$.

LEMMA 1.2.
If $b|a$ and $a\ne0$, then $|b|\le|a|$.
Proof.
Indeed, by definition of divisibility we have $a=bc$ for some integer $c$; in particular, both $b$ and $c$ are nonzero since $a\ne0$. Since $c\in\mathbb{Z}$ and $c\ne0$, we have $|c|\ge{1}$. And then $$|a|=|b|\cdot|c|\ge{|b|}\cdot{1} = |b|$$ as claimed.

Divisibility defines an order relation on the set of nonnegative integers. What Lemma 1.2 says is that, to some extent, this "new" order relation is compatible with the ordinary one $\le$.

Well-ordering principle.
Every nonempty set of nonnegative integers contains a least element.

THEOREM 1.3. Division with remainder.
Let $a,b$ be integers, with $b\ne0$. Then there exist a unique $q\in\mathbb{Z}$ and a unique $r\in\mathbb{Z}$ such that $a=bq+r$ with $0\le{r}\lt|b|$.
Proof.
We can assume that $b\gt{0}$.
Indeed, if $b\lt{0}$, we can apply the statement to $-b\gt{0}$ and then flip the sign of $q$ after the fact. Switching the sign of $b$ does not change $|b|$, so the condition on $r$ is unchanged.
Assume then that $b\gt{0}$, and consider all integer linear combinations of $a$ and $b$ of the form $a-bx$ with $x\in\mathbb{Z}$. Note that there are nonnegative integers of this type: if $a$ is itself nonnegative, then $a-b\cdot0=a$ is nonnegative; and if $a$ is negative, then $a-ba=a(1-b)$ is nonnegative because $1-b\le0$. Therefore, the set $$S=\{a-bx\ |\ x\in\mathbb{Z}: a-bx\ge0\}$$ is a nonempty set of nonnegative integers.
By the well-ordering principle, it contains a least element $r$: that is, there is some $x=q$ such that $r=a-bq\ge0$ is smaller than any other nonnegative number of the form $a-bx$. And these $q$ and $r$ are the unique numbers whose existence is claimed in the statement.
Indeed: by construction we have that $a=bq+r$ and that $r\ge0$; we have to verify that (1) $r\lt{b}$ (note that $|b|=b$ since we are assuming $b\gt0$) and (2) the numbers $q$ and $r$ are unique with these properties.
For (1), argue by contradiction. If we had $r\ge{b}$, then we would have $r-b\ge0$; and $$0\le{r-b}=(a-bq)-b=a-b(q+1).$$ This would show that $r-b$ is an element of $S$, since it is a nonnegative integer linear combination of $a$ and $b$. But that is a contradiction, since $r-b\lt{r}$ while $r$ was chosen to be the least element of $S$. Since $r\ge{b}$ leads to a contradiction, we can conclude that $r\lt{b}$ as we claimed.
For (2), assume that $q',r'$ are also integers satisfying the requirement spelled out in the theorem: that is, $a=bq'+r'$, and $0\le{r'}\lt{b}$. We will show that necessarily $q'=q$ and $r'=r$ (this is what we mean by stating that $q$ and $r$ are unique).
Since $a=bq+r=bq'+r'$, we have $$b(q-q')=r'-r.\qquad\qquad\qquad(1.1)$$
Now, since both $r$ and $r'$ are in the interval $[0, b-1]$, their difference cannot exceed $b-1$. In other words, $|r'-r|\le{b-1}$, and in particular $|r'-r|\lt|b|$. But (1.1) shows that $b$ is a divisor of $r'-r$; if $r'-r\ne0$, then by Lemma 1.2 we would necessarily have $|b|\le|r'-r|$, contradicting the fact that $|b|>|r'-r|$ we observed a moment ago. The only possibility then is that $r'-r=0$. This shows that $r'=r$, and further $b(q-q')=0$. Since $b\ne0$, this implies that $q-q'=0$, and it follows that $q'=q$.

DEFINITION 1.5. Greatest common divisor.
Let $a,b\in\mathbb{Z}$. We say that a nonnegative integer $d$ is the greatest common divisor of $a$ and $b$, denoted $gcd(a, b)$ or simply $(a, b)$, if:
(i) $d | a$ and $d | b$;
(ii) if $c | a$ and $c | b$, then $c | d$.

If at least one of $a$ and $b$ is not $0$, then $d = gcd(a, b)\ne 0$. Indeed, $d$ must divide both $a$ and $b$ according to the requirement (i) in the definition (this makes $d$ a common divisor), so $d$ cannot be $0$ unless both $a$ and $b$ are $0$ - keep in mind that $0$ only divides $0$.
The requirement (ii) in the definition says that if $c$ is also a common divisor, then it must divide $d$. By Lemma 1.2, this implies that $|c|\le|d|$ if $d\ne 0$, and hence $c\le{d}$ since $d$ is nonnegative. Therefore $d$ is the greatest (as in largest) common divisor of $a$ and $b$ if $a$ and $b$ are not both $0$.
Why requirement (ii) is not something like:
(ii) if $c | a$ and $c | b$, then $c\le{d}$,
which would seem to justify the terminology greatest common divisor in a more direct way. The main reason is that when we recast this notion for more general rings, the relation of divisibility will be available, while the (perhaps) simpler relation $\le$ will not. Also, Definition 1.5 makes good sense for all possible integers $a, b$, including the case $a = b = 0$: in this case, since every number divides $0$, the second requirement says that every integer should divide $gcd(0, 0)$. Since $0$ is the only number divisible by every integer, this tells us that $gcd(0, 0) = 0$ according to Definition 1.5. Even in this case, $gcd(a, b)$ is greatest; but with respect to the divisibility relation rather than the ordinary order relation $\le$.

THEOREM 1.8.
Let $a,b$ be integers. Then the greatest common divisor $d = gcd(a, b)$ is an integer linear combination of $a$ and $b$. That is, there exist integers $m$ and $n$ such that $d = ma + nb$.
In fact, if $a$ and $b$ are not both $0$, then $gcd(a, b)$ is the smallest positive linear combination of $a$ and $b$.
Proof.
If $a = b = 0$, then $gcd(a, b) = gcd(0, 0) = 0$, which is a linear combination of $a$ and $b$ (because $0 = 1 · 0 + 1 · 0$). Therefore, we may assume that $a$ and $b$ are not both $0$. Consider all linear combinations $ma + nb$ of $a$ and $b$. I claim that some of them are positive. Indeed, take $m = a, n = b$; then $ma + nb = a^2 + b^2$ , and this number is positive since $a$ and $b$ are not both $0$.
Therefore, the set $$S = \{ma + nb\ |\ m\in\mathbb{Z}, n\in\mathbb{Z}, ma + nb \gt 0\}$$ is nonempty. This is the standard setting to apply the well-ordering principle: since $S$ is a nonempty set of nonnegative integers, it must have a least element. Let $d$ be this element, and let $m$ and $n$ be integers such that $d = ma + nb$. That is, $d$ is the smallest positive linear combination of $a$ and $b$. We are going to verify that $d$ is the $gcd$ of $a$ and $b$, and this will prove the theorem.
Since $d\in{S}$, then $d \gt 0$: so $d$ is nonnegative and not equal to $0$. In order to prove that $d$ is the $gcd$ of $a$ and $b$, we have to prove that
(i) $d | a$ and $d | b$;
(ii) if $c | a$ and $c | b$, then $c | d$.
If $a = 0$, $d|a$ is automatic. If $a \ne 0$, we can use division with remainder: by Theorem 1.3, we know that there exist integers $q$ and $r$ such that $a = dq + r$, with $0 \le r \lt d$. What can we say about $r$? Note that $$r = a - dq = a - (ma + nb) q = a (1 - mq) + b (-nq) :$$ this shows that $r$ is a linear combination of $a$ and $b$. Can it be an element of $S$? No! Because $r \lt d$, and $d$ has been chosen to be the smallest element of $S$. But then $r$ cannot be positive, since $r$ is a linear combination of $a$ and $b$, and $S$ contains all positive linear combinations of $a$ and $b$. Since $r \ge 0$ and $r$ is not positive, it follows that $r = 0$. This proves that $a = dq$, showing that $d|a$.
By essentially the same argument, we can deduce that $d|b$. This takes care of (i).
We still have to prove (ii). Suppose we have a common divisor $c$ of $a$ and $b$: $c|a$ and $c|b$. Then we have $a = uc, b = vc$ for some integers $u$ and $v$. This gives $$d = ma + nb = m (uc) + n (vc) = (mu + nv) c$$ and proves that $c|d$, as we needed.

COROLLARY 1.9.
Let $a, b$ be integers. Then $gcd(a, b) = 1$ if and only if $1$ may be expressed as a linear combination of $a$ and $b$.
Proof.
($\Rightarrow$) If $gcd(a, b) = 1$, then the number $1$ may be expressed as a linear combination of $a$ and $b$ by Theorem 1.8.
($\Leftarrow$) If $1$ may be expressed as a linear combination of $a$ and $b$, then $1$ is necessarily the smallest positive linear combination of $a$ and $b$, because $1$ is the smallest positive integer. It follows that $gcd(a, b) = 1$, again as a consequence of Theorem 1.8.

DEFINITION 1.10. Relatively prime integers.
Let $a,b\in\mathbb{Z}$. We say that $a$ and $b$ are relatively prime if $gcd(a, b) = 1$.

COROLLARY 1.11.
Let $a, b, c$ be integers. If $a|bc$ and $gcd(a, b) = 1$, then $a|c$.
Proof.
By Theorem 1.8, $1 = ma + nb$ for some integers $m$ and $n$. Multiply by $c$: $$c = 1 \cdot c = (ma + nb) c = mac + nbc = (mc) a + n (bc).$$ Now the hypothesis tells us that $a|bc$, so $bc = ra$ for some integer $r$. Then $$c = (mc) a + n (ra) = (mc + nr) a,$$ and this shows that $a|c$ as needed.

Euclidean algorithm
Given two integers $a$ and $b$, we may assume that $b\ne0$ (if $b = 0$, then $gcd(a, b) = a$, so there is nothing to compute). We can then apply division with remainder repeatedly, like this: $$a = b q_0 + r_0 \qquad 0 \le r_1 \lt b$$ $$b = r_0 q_1 + r_1 \qquad 0 \le r_1 \lt r0$$ $$r_0 = r_1 q_2 + r_2 \qquad 0 \le r_2 \lt r1$$ $$r_1 = r_2 q_3 + r_3 \qquad 0 \le r_3 \lt r2$$ $$···$$ $$r_{n-2} = r_{n-1} q_n + r_n \qquad 0 \le r_n \lt r_{n-1}$$ $$r_{n-1} = r_n q_{n+1} + 0$$ We keep going so long as the remainder of the last division is not 0. Since at each step the remainder decreases and remains nonnegative, at some point it must become $0$. Last nonzero reminder ${r_n} = $gcd(a,b)$.

LEMMA 1.15.
Let $a, b, q, r$ be integers, with $b \ne 0$, and assume that $a = bq + r$. Then $gcd(a, b) = gcd(b, r)$.
Proof.
It is enough to prove that the pairs $a$ and $b$, $b$ and $r$ have the same common divisors: then the largest among the common divisors will be the $gcd$ of both $(a, b)$ and $(b, r)$, proving the statement.
Let $c$ be a common divisor of $a$ and $b$: $c|a$ and $c|b$. Then $a = cs, b = ct$ for some integers $s$ and $t$. It follows that $$r = a - bq = cs - ctq = c (s - tq),$$ and this shows that $c|r$. Since we already have that $c|b$, we see that $c$ is a common divisor of $b$ and $r$.

THEOREM 1.14. Euclidean algorithm.
Proof.
By Lemma 1.15: $gcd(a, b) = gcd(b, r_0) = gcd(r_0, r_1) = \dots = gcd(r_{n-1}, r_n) = gcd(r_n, 0) = r_n$.

If necessary, one can chase the Euclidean algorithm backwards and find integers $m, n$ such that $gcd(a, b) = ma + nb$.
