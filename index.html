Глава 1. Введение

§1.1 Числовой континуум

Определение. Неравенство треугольника.
$$|a+b|<=|a|+|b|$$

Док-во:

Рассмотрим два случая:

1) Если $a+b\ge0$, то имеем $a+b\le|a|+|b|$, что сразу следует из суммы неравенств $a\le|a|$ и $b\le|b|$.

2) Если $a+b\lt0$, то имеем $-(a+b)\le|a|+|b|$, что, снова, сразу следует из суммы неравенств $-a\le|a|$ и $-b\le|b|$.

ч.т.д.

Следствие: $|a_1+a_2+...+a_n|\le|a_1|+|a_2|+...+|a_n|$.

Док-во:

Докажим для случая $n=3$.

Дважды применим неравенство треугольника: $|a+b+с|=|(a+b)+с|\le|a+b|+|с|\le|a|+|b|+|с|$.

Аналогично доказывается и для всех n.

Definition. A partition of a set.

A partition of set S is a set of disjoint nonempty subsets of S, whose union is S

Definition 1.2. The quotient of the set with respect to the equivalence relation ∼.

The quotient of the set S with respect to the equivalence relation ∼ is the set

$$S/∼ := \mathcal{P}_∼$$

of equivalence classes of elements of S with respect to ∼.
