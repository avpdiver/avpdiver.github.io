Введение

Типы задач ML


Задачи классификации (classification)
  • $Y=\{-1,1\}$ - бинарная классификация
  • $Y=\{1,\dots,M\}$ - классификация $M$ непересекающихся классов
  • $Y=\{0,1\}^M$ - классификация $M$ пересекающихся классов

Задачи восстановления регрессии (regression)
  • $Y=\mathbb{R}$
  • $Y=\mathbb{R}^n$

Задачи ранжирования (ranking)
  • $Y$ - конечное упорядоченное множество

Задачи обучения без учителя (unsupervised learning)
  • $Y=\emptyset$ - ответов нет, требуется что-то делать с самими объектами



Предсказательная модель

Модель (predictive model) - параметрическое семейство функций $$A=\{a(x,w)\ |\ w \in W\}$$ где $a:X \times W \to Y$ - фиксированная функция, $W$ - множество допустимых значений параметра $w$.
Пример
Линейная модель с параметром $w=(w_1,\dots,w_n)\in\mathbb{R}^n$: $$a(x,w)=\sum_{j=1}^nw_jf_j(x) - \text{регрессия и ранжирование}$$ $$a(x,w)=sign\left(\sum_{j=1}^nw_jf_j(x)\right) - \text{бинарная классификация}$$


Функция потерь $\mathbb{L}(a, x)$ - величина ошибки алгоритма $a\in{A}$ на объекте $x\in{X}$.
Функции потерь для задач классификации
$\mathbb{L}(a, x)=[a(x)\neq{y(x)}]$ - индикатор ошибки.
Функции потерь для задач регрессии
$\mathbb{L}(a, x)=|a(x)-y(x)|$ - абсолютное значение ошибки.
$\mathbb{L}(a, x)=(a(x)-y(x))^2$ - квадратичная ошибка.

Эмпирический риск - функционал качества алгоритма $a$ на $X^l$: $$Q(a,X^l):=\frac{1}{l}\sum_{i=1}^l\mathbb{L}(a, x_i)$$

Метод минимизации эмпирического риска (Empirical Risk Minimization, ERM): $$\mu(X^l) = \arg\min_{a\in{A}}Q(a,X^l)$$
Задача регрессии:
$Y=\mathbb{R}$;
$n$ числовых признаков $f_j(x), j=1,\dots,n$;
линейная модель регресии $g(x,w)=\sum_{i=1}^n w_j f_j(x),\ w\in\mathbb{R}^n$;
квадратичная ф-ция потерь $\mathbb{L}(a,x)=(a(x)-y(x))^2$.
Метод наименьших квадратов (частный случай ERM): $$\mu(a,X^l):=\arg\min_w\sum_{i=1}^l(g(x_i,w)-y_i)^2$$

Эмпирический риск на тестовых данных (hold-out): $$HO\left(\mu,X^l,X^k\right):=Q\left(\mu\left(X^l\right),X^k\right)\to\min $$ Скользящий контроль (leave-one-out), $L=l+1$: $$LOO\left(\mu,X^L\right):=\frac{1}{L}\sum_{i=1}^L\mathbb{L}\left(\mu\left(X^L\setminus\{x_i\}\right), x_i\right)\to\min$$ Кросс-проверка (cross validation), $L=l+k$: $$CV\left(\mu,X^L\right):=\frac{1}{|P|}\sum_{p\in{P}}Q\left(\mu\left(X_p^l\right), X_p^k\right)\to\min,$$ где $P$ - множество разбиений $X^L=X_p^l\sqcup{X_p^k}$.
